# Development Guidelines

Before planning or making changes, read these repository-root files in order
and follow their instructions:

1. `README.md`
2. `DEVELOPMENT.md`
3. `DEVELOPMENT_SHARED.md`
4. `DEVELOPMENT_AGENT.md`
5. `DEVELOPMENT_SHARED_AGENT.md`

Treat the files as one instruction set. Where they define precedence, apply the
more specific guidance as directed.
